# 평면 프레임 빌드

PX4는 정상적인 비행기, 날기 날개, 역 V 자형 비행기 등 모든 상상할 수있는 평면 형상을 지원합니다. 지원되는 구성의 전체 세트는 [기체 프레임 참고문헌> 비행기](../airframes/airframe_reference.md#plane)에서 확인할 수 있습니다.

이 섹션에는 여러 평면 프레임을 조합하고 구성하기위한 빌드 로그 / 지침이 들어 있습니다.
