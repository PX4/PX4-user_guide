# 고정익 조립

PX4는 일반 평면, 날아 다니는 날개, 반전 된 V- 테일 평면 등 상상할 수있는 모든 평면 형상의 고정익을 지원합니다. 지원되는 전체 세트는 [기체 프레임 참고문헌> 비행기](../airframes/airframe_reference.md#plane)에서 확인하십시오.

다양한 기체들의 조립 방법과 설정 방법을 설명합니다.
