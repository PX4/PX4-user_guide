# 로버 탐사선(UGVs)

PX4는 무인지상기체 (UGVs)의 기능을 제공합니다. 지원되는 기체의 종류는 [기체 정의서> 로버](../airframes/airframe_reference.md#rover)에서 확인하십시오.

UGV 프레임 조립 방법과 설정 방법을 제공합니다.
