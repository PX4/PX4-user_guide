# 고급 설정

이 섹션에는 기체 튜닝과 덜 일반적인 센서 및 주변 장치들의 고급 설정 방법을 설명합니다.

:::tip
이 항목은 새로운 기체를 제작하거나 표준 프레임을 수정하는 경우에 유용합니다.
:::tip
이 항목은 새로운 기체를 제작하거나 표준 프레임을 수정하는 경우에 유용합니다. 
:::

기체의 적합한 표준 설정은 [기본 설정](../config/README.md)편을 참조하십시오.
