# 기체 프레임/기체 제작

PX4는 멀티콥터, 고정익, 수직 이착륙기(VTOL), 자동차 등 여러 종류의 운송체에 적용됩니다. 지원되는 운송체에 관련된 설정은 [기체 프레임 정의서](../airframes/airframe_reference.md)편을 참고하십시오.

이 절에서는 다양한 [비행 컨트롤러](../flight_controller/README.md)를 다양한 기체에 설치하는 방법을 설명합니다.
