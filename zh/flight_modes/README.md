# 飞行模式

*飞行模式* 定义自动驾驶仪如何响应用户输入并控制飞机移动。 下表总结了固定机翼和直升机的飞行模式（[table键在下面](#key)）。 请注意，这是 "高级" 默认行为，可能因飞机参数而异。 链接的主题（侧边栏）提供了有关各个模式的更详细信息，包括它们的调优参数。

> **Tip** [开始 > 航班管理](../getting_started/flight_modes.md) 中对所有飞行模式的 *初学者友好* 解释。
