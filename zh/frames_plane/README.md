# 飞机框架构建

PX4支持各种可以想象到的飞机几何结构，包括普通飞机、飞行翼、倒V型等。 The complete set of supported configurations can be seen in [Airframes Reference > Plane](../airframes/airframe_reference.md#plane).

本节包含用于组装和配置多个平面机架的构建日志和说明。
